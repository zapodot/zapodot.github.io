Some projects I am currently working on:

  • Embedded DB Junit rule - JUnit rule for running an in-memory H2 database in your tests
  • Embedded LDAP Junit rule - JUnit rule for running an in-memory UnboundID LDAP Server in your tests
  • Hystrix DropwWizard bundle - a DropWizard bundle that enables reporting for Hystrix
  • Jackson Databind Java Optional - Support for Java Optional with Jackson's Databinding module
  • Akka JUnit Rules - JUnit rules for simplifing testing of Akka actors
  • I am also a co maintainer for the Constretto project

Here are a few projects that I have previously worked on, but have since abandoned (left here for educational purposes):

  • Play Constretto - A Play! Framework 2.1 plugin for enabling the Constretto configuration tool to be used in conjunction with the standard Play configuration mechanism
